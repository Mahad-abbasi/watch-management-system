# Watch Management System

A full-stack Watch Management System developed to manage watch inventory, customers, sales, and administrative operations.

## Tech Stack

### Frontend
- React.js
- JavaScript
- CSS

### Backend
- Java Spring Boot
- REST APIs
- Spring Data JPA

### Database
- SQL Server

## Features

- User Authentication
- Watch Inventory Management
- Customer Management
- Sales Management
- Product Tracking
- Admin Dashboard

## Project Structure
watch-management-system
│
├── BACKEND SPRING BOOT
│
└── FRONTEND REACT

## Installation

### Backend

Navigate to:
BACKEND SPRING BOOT


Run:


mvn spring-boot:run


### Frontend

Navigate to:


FRONTED REACT


Install dependencies:


npm install


Run:
npm start
